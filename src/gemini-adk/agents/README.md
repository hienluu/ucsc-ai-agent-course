## ADK agent examples

### Command to run them
- Make sure you are in the "agents" folder

#### To run them in terminal
```bash
adk run basic_agent.py
```

```bash
adk run agent_doubleword.py
```

- To exit the console, type "exit"

#### To run them in ADK web UI
- if you are already in the "agents" folder

```bash
adk web .
```

- if you are in the "gemin-adk" folder
```bash
adk web agents
```

- to exit, type ctrl-c keys
